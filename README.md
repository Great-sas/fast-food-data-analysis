# Fast Food Nutrition Analysis

Project Description
Analysis of nutritional data from major fast-food chains.

Quick Start
1. Download `fast_food_analysis.ipynb`
2. Install: `pip install pandas matplotlib seaborn jupyter`
3. Run: `jupyter notebook fast_food_analysis.ipynb`

Key Insights
- McDonald's has the largest menu (328 items)
- Found correlation between protein and calories
- Identified healthiest options

Technologies
- Python, Pandas, Matplotlib, Seaborn

Dataset
FastFoodNutritionMenuV3.csv - 1,147 menu items from 6 restaurants
